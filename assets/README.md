# Assets for Risk0 Bot

Place your visual assets here:

- base_template.png  (recommended 1080x1350)
- risk0_logo.png     (PNG with transparent background)
- Inter-Regular.ttf  (font)

Optional: add bookmaker icons like `book_leovegas.png`, `book_betvictor.png` (e.g., 96x96) if you extend the card.

Environment overrides:
- RISK0_BASE_TEMPLATE
- RISK0_LOGO
- RISK0_FONT

If assets are missing or invalid, the bot falls back to a simple solid background and default font.
